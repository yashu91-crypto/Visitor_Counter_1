# Visitor Counter Using ESP32 and IR Sensor

## Overview
This project is a Visitor Counter System built using ESP32 and an IR Sensor. The system detects visitors automatically and maintains a count. A buzzer provides alerts, and a relay module can be used to control external devices.

## Components Used
- ESP32
- IR Sensor
- 4-Channel Relay Module
- Buzzer
- Breadboard
- Jumper Wires

## Features
- Automatic visitor detection
- Visitor counting
- Buzzer alert system
- Relay-based control
- Real-time monitoring
