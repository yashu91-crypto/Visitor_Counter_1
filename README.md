# Visitor Counter Using ESP32 and IR Sensor

## Overview
This project is a Visitor Counter System built using ESP32 and an IR Sensor. The system detects visitors automatically and maintains a count. A buzzer provides alerts, and a relay module can be used to control external devices.

## Components Used
- ESP32
- IR Sensor
- 4-Channel Relay Module
- Buzzer
- Breadboard
- Jumper Wires

## Features
- Automatic visitor detection
- Visitor counting
- Buzzer alert system
- Relay-based control
- Real-time monitoring

## Skills Learned
- ESP32 Programming
- Embedded Systems
- Sensor Interfacing
- Circuit Design
- IoT Fundamentals[Visitor_Counter.pdf (1).pdf](https://github.com/user-attachments/files/29172667/Visitor_Counter.pdf.1.pdf)

## Working Principle
The IR sensor detects a visitor passing through its sensing range. The ESP32 processes the sensor signal, updates the visitor count, and activates the buzzer. The relay module can be used for additional automation tasks.
